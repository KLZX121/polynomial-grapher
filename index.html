Polynomial Grapher

Input

String Input

Format ax^n+bx^(n-1)+...+px+q with no spaces and no parentheses.

For example: 2x^3+3x+1 -5x^5+x^3-7x^2

Parameter Input

Format Degree: Integer of max exponent in polynomial Coefficients: Comma separated values of coefficients
For example: -5x^5+x^3-7x^2 Degree: 5 Coefficients: -5,0,1,-7,0,0

Graph Settings

Enter domain as two comma separated values of x min and x max. For example: -10,10

Standard Plot

Step is the difference between each plotted x value. Produces a uniform distribution of points. The lower the step, the more accurate the function. Enter a integer or decimal value. For example: Domain = -2, 2 Step = 0.1 would plot all points with x values -2,-1.9,-1.8,-1.7,...,1.7,1.8,1.9,2

Smart Plot

Sample Rate is the rate at which the program will check for viable points. The lower the value, the more accurate the plotting will be. For example: Values of 0.01 to 0.1 are generally ideal

Precision is the density of points to be plotted. A lower precision will equal a higher density and number of points, increasing accuracy. For example: Values of 0.01 to 1 are generally ideal
